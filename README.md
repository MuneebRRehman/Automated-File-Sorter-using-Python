# 📂 Automated File Sorter using Python

This project is a simple Python automation script that organizes files in a directory into separate folders based on their file type.

## 🚀 Features

- Automatically creates folders if they don’t exist
- Sorts files into:
  - CSV files
  - PNG images
  - Text files
- Uses Python file handling and OS automation
- Helps keep directories clean and organized

## 🛠️ Technologies Used

- Python
- os module
- shutil module

## 📁 How It Works

1. Specify the target directory path
2. The script checks for required folders
3. It creates folders if they are missing
4. Files are moved into respective folders based on their extension

## 🧾 Supported File Types

- `.xlsx` → CSV folder  
- `.txt` → Text folder  
- `.png` → Image folder
